# AI Resume Parser Using NLP

## 📌 Project Overview

The AI Resume Parser is a Natural Language Processing (NLP) based application that automatically analyzes resumes, extracts technical skills, and generates a resume score. The system helps recruiters and organizations quickly evaluate candidate profiles by identifying relevant skills and presenting the results in an interactive web interface.

---

## 🎯 Objective

- Automate resume screening and analysis.
- Extract technical skills from resumes.
- Generate a resume score based on detected skills.
- Visualize skill analysis using graphs.
- Provide a simple and user-friendly interface using Streamlit.

## 🧠 Algorithms Used

### Natural Language Processing (NLP)
Used for cleaning and preprocessing resume text.

### Stop Word Removal
Removes common English words that do not contribute to skill analysis.

### Rule-Based Skill Extraction
Identifies technical skills using a predefined skills database.

### Resume Scoring Algorithm
Assigns weighted scores based on detected skills and generates a score between 0 and 10.

```bash

```

### 2. Open Project Folder

```bash

```

### 3. Install Required Libraries

```bash

```

### 4. Run Streamlit Application

```bash
streamlit run app.py
```
